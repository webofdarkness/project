## Description
A **d10-based system** designed to manage character sheets for both **Storytellers** and **Players**, inspired by the tabletop RPG *World of Darkness* and its many dark chronicles.

This project aims to provide a **self-hosted** environment where players and storytellers can experience and track their campaigns in local servers, preserving the tone, mystery, and atmosphere true to the original spirit of the *World of Darkness*.

- **Character Sheet System:**  
  Storytellers can create and assign blank sheets to players.

- **Chronicle Tracking:**  
  Sessions and characters are persisted locally, allowing long-term management of ongoing stories.

  - A dynamic **Patrons List** available inside the system.
